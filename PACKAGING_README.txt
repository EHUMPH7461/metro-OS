METRO OS WINDOWS APP BUILD

Fastest method:
1. Double-click BUILD_WINDOWS_APP.bat.
2. Wait for the build to finish.
3. The RELEASE folder opens automatically.
4. Run the Metro OS installer, or use the portable EXE.

The installer creates a desktop and Start Menu shortcut.
Node.js 22 LTS is required only to build the application. After installation,
Metro OS opens normally without npm commands or Command Prompt.
